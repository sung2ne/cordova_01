# 코도바 실행 예제

## Node.js 설치

https://nodejs.org/ko/

자신의 PC에 알맞은 파일 다운로드 후 설치

## 코도바 설치

```bash
$ npm install -g cordova
```

## 프로젝트 생성

```bash
$ cordova create 프로젝트
```

## 프로젝트로 이동

```bash
$ cd 프로젝트
```

## 안드로이드 플랫폼 추가

```bash
$ cordova platform add android
```

## 앱 실행

```bash
$ cordova run android
```
